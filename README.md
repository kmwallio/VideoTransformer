# VideoTransformer

This will take one source video and overlay it atop another video.  It does this for every frame.

## Installation

Just download and run `perl transform.pl`

## Requirements

* mplayer
* ffmpeg
* Perl
* GTK-Perl
* Perl Modules \(in [CPAN](http://www.cpan.org/ "CPAN")\):
	* GD
	* Gtk2 (GTK-Perl)
	* Gtk2::Ex::MPlayerEmbed
	* Video::FrameGrab
	* GStreamer
